点击白色区域

  • 事件冒泡：事件开始由最具体的元素开始接受，然后逐级向上传播到较为不具体的节点
  • 事件捕获：事件开始由最不具体的元素开始接受，然后逐级向上传播到较为具体的节点
  • 目标元素：event.target
  • 停止事件冒泡 if ( e && e.stopPropagation ){ //因此它支持W3C的stopPropagation()方法 e.stopPropagation(); }else{ //否则，我们需要使用IE的方式来取消事件冒泡 window.event.cancelBubble = true; return false; }
  • 阻止浏览器的默认行为 if ( e && e.preventDefault ){ //阻止默认浏览器动作(W3C) e.preventDefault(); }else{ //IE中阻止函数器默认动作的方式 window.event.returnValue = false; return false; }
